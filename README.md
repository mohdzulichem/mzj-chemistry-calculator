# MZJ Chemistry Web Calculator (Enhanced)

Lightweight, mobile-friendly chemistry calculators with **UiTM/MZJ banner**, **Malay/English toggle**, and **Smart Chemistry via AI** branding.

**Modules:** Molar Mass • Molarity • Dilution • pH (strong acid/base) • Ideal Gas Law • Mass–Moles–Molar Mass

## Deploy on GitHub Pages
1. Create a new repository (e.g., `mzj-chemistry-calculator`).
2. Upload `index.html` and the entire `assets/` folder.
3. **Settings → Pages → Source:** main branch /root → **Save**.
4. Your site: `https://<username>.github.io/mzj-chemistry-calculator/`
